Definicion y ejemplo de destructuracion de objetos(de 1 nivel y de haasta 3 niveles)

Es una característica que permite extraer propiedades específicas de un objeto y asignarlas a variables individuales de forma más simplificada y con una sintaxsis especial de desestructuración. Es útil cuando se trabaja con grandes objetos o cuando solo se necesitan algunas propiedades.

Definicion y ejemplo de freezear un objeto

Se utiliza para congelar un objeto, lo que significa que no se pueden realizar cambios en sus propiedades, ya sea agregar o eliminar las que ya tiene existentes, es decir, es inmutable.

Definicion y ejemplo de sellar un objeto

Cuando se sella un objeto no se pueden agregar nuevas propiedades o eliminar otras que estén en el objeto literal, pero se pueden modificar las propiedas ya existentes.

Definicion y ejemplo de unir o sumar dos objetos

Se puede univr o combinar dos objetos creando uno nuevo que contenga las propiedades de los objetos originales. Se utilizará object.assign.

Definicion y uso practico de palabra reservada this.

Se refiere a un objeto que represente el contexto en el cual se está ejecutando el código en el momento actual.
